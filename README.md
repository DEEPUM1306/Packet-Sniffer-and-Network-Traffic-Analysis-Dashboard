# 📡 Packet Sniffer & Traffic Analysis Dashboard

A Streamlit-based Network Packet Sniffer that captures live packets or analyzes uploaded PCAP files. The application provides interactive visualizations and detects suspicious traffic patterns.

## Features

- Live Packet Capture
- Upload PCAP Files
- Protocol Filtering
- Traffic Dashboard
- Source/Destination IP Analysis
- Port Distribution
- Packet Size Analysis
- Suspicious Traffic Detection
- Download CSV Report

## Technologies Used

- Python
- Streamlit
- Scapy
- Pandas
- Plotly

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
